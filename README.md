# ft_printf

This project is an introduction to variadic functions and the macros related to them

## <stdarg.h>
+ va_arg
+ va_start
+ va_copy
+ va_end

Basic implementation of printf

### External functions allowed:
+ malloc
+ free
+ write

Handles %(c,s,p,u,d,i,x,X,%)
